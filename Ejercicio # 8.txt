Efectos en cada caja:

.caja1:hover{
  transform: translateX(100px);
  transition: 2s;
}

.caja2:hover{
  transform: rotate(360deg);
  transition: 2s;
}

.caja3:hover{
  transform: rotateX(360deg);
  transition: 2s;
}

.caja4:hover{
  transform: rotateY(360deg);
  transition: 2s;
}

TranslateX (se desplaza en el eje X)
Rotate (x) (y)  rota en ambos ejes 
Transition: tiempo en que se ejecuta cada efecto
